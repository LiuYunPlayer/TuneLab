# DraggableNumberBox 控件验证（独立用例）

只验证新增的可拖拽无界数值框控件（`DraggableNumberBox` / `DraggableNumberBoxConfig` / `IDragResponse`）。范围仅限该控件本身，不涉及音素编辑器其余待做部分。

## 前置

- 已 build + pack + install `v1-voice`（包名 **V1 Test Voice**）。
- 测试夹具：`V1 Test Voice` 引擎在每个音素上声明了两条属性——
  - `accent`：滑条（`SliderConfig`，0..1），作对照。
  - **`Offset (ms)`：可拖拽数值框（`DraggableNumberBoxConfig`，无界）**，本次被测对象。线性灵敏度 0.5/px、1 位小数显示。

## 进入被测面板

1. 打开 TuneLab，新建工程，给 part 选 **V1 Test Voice** 下任一声库（Alice / Bob / Carol）。
2. 画几个有歌词的音符（如 `ka`、`skat`），等待合成（约 1s）后音素带出现音素。
3. 选中一个含音素的音符 → 右侧 note 属性栏 **Phoneme** 页签：每个音素一行，行内含 `accent` 滑条与 **`Offset (ms)`** 数值框。

## 用例

| # | 操作 | 期望 |
|---|---|---|
| 1 | 在 `Offset (ms)` 框上**横向拖动** | 数值实时变化；右拖变大、左拖变小；无上下界（可拖到很大 / 负值）。 |
| 2 | 拖动时按住 **Shift** | 同样位移下变化量明显变小（精调，约 1/4）。 |
| 3 | 纵向拖动 | 默认水平轴，纵向不改值（仅横向驱动）。 |
| 4 | **双击**该框 | 原地出现输入框、全选；键入数字后回车 / 点击别处 → 提交为该值。 |
| 5 | 双击输入非法文本（如 `abc`）回车 | 不改值，回退显示原值。 |
| 6 | 一次拖动后 **Ctrl+Z** | 整段拖动作为**一个撤销步**回退到拖动前的值（非逐帧多步）。 |
| 7 | 双击键入一次后 **Ctrl+Z** | 一步回退到键入前的值。 |
| 8 | 多选多个同结构音符的 note，改一个 `Offset` | 扇出到所选各音素；值不一致时框显 `-`（多值态），拖动 / 键入后统一。 |
| 9 | 改 `Offset` 后存盘、重开工程 | 值持久（per-phoneme 属性走既有持久链路）。 |
| 10 | 鼠标悬停该框 | 光标变为左右拉伸提示（可拖暗示）。 |

## 已知非目标（本次不验证）

- 音素符号编辑、音素增删、字段层级（权重沉底）——属音素编辑器后续阶段，未接线。
- `DragResponse.Exponential` / `Custom`、`Min`/`Max` 单双边钳——本夹具只声明了线性无界一种，其余路径靠单测 / 后续夹具覆盖。
