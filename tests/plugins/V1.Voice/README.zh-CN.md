## 快速开始

这个声源在使用前需要一点配置：

1. 打开 **设置 → 扩展**，选择一个声库。
2. 在任意轨道上新建一个*声源片段*。
3. 输入歌词——每个音符会生成对应音素。

## 参数

| 参数   | 含义         | 默认值 |

| speed  | 播放速度     | 1.0    |
| pitch  | 全局音高偏移 | 0      |
| breath | 气声量       | 0.3    |
