# 参数区值轴标度（INormalizedScale）接通测试清单

本清单只覆盖「参数区值轴改经 config 标度（`INormalizedScale`）+ 离散标度求值量化」这次改动**受影响的范围**，
不复测已通过的参数区/属性 lane 基线。改动要点（被测对象）：

- `AutomationConfig` 新增 `Create(INormalizedScale)` 重载（如 `NormalizedScale.Integer(min, max)` = 整数轨）；
- 参数区连续轨/分段轨的**绘制、拖锚点、双击插锚**全走标度取值（离散标度 ⇒ 写入值吸附，拖动定位不跳变）；
- **离散标度求值量化（本次核心）**：宿主在渲染 + 合成快照把 Hermite 连续输出投影回标度——离散轨曲线呈**阶梯**、
  插件 `Evaluate` 读到的最终值**处处落格**（不止锚点）；这条覆盖 load/preset/回喂/undo 等一切绕过操作层的路径。
  线性轨投影 = 纯范围钳位（越界值钳回 `[min,max]`、无视觉改变）；
- 属性 lane（钉选 note/phoneme 属性）改为直接携带 `SliderConfig.Scale` ——整数滑条属性钉上参数面板后拖写同样吸附；
- 框选锚点的值轴比较改在归一化域进行（离散标度下选框不跳格，行为对线性轨不变）；
- 侧栏默认值滑条本就消费 `Scale`，不在本次范围。

设计边界（非缺陷）：曲线是「Hermite 连续插值 → 投影落格」，故阶梯的**跳变位置**取决于平滑曲线穿越格中点处，
不由用户精确指定——整数轨足够，真正要精确区间边界的「二值开关」应另用分段轨（step 语义），不在本轨模型内。

## 前置

1. 已为你 build + pack 好测试插件：`tests/tlx/v1-voice.tlx`（重新安装覆盖旧版）。
2. 新建 track 选 **Alice (V1 Test)**（或 Bob/Carol），建 MidiPart 画几个 note。
3. 新增夹具：
   - 参数栏新增连续轨 **Semitone (Int)**（整数吸附标度 −12..12，默认 0，浅绿色，0 位小数显示）；
   - 参数栏新增连续轨 **LogFreq (Hz)**（对数轴 Custom 标度 20..20000Hz，默认 440，青色，显示 "440 Hz"）；
   - note 属性新增 **Steps (Int)**（`SliderConfig.Integer(0, 0, 8)`），用于钉选 lane 验证。
4. 日志在 `%APPDATA%/TuneLab/Logs/`。

## 用例

### A. 整数轨绘制吸附（画笔）
1. 参数栏切到 **Semitone (Int)**，用画笔拖一条斜线 → 曲线呈台阶感（值只落整数位）；
   按住 **Ctrl** 定值横扫 → 整条线锁在按下时的整数值。
2. 对照：切到 **Growl**（线性轨）随手画 → 行为与改动前一致（平滑、无台阶）。

### B. 整数轨锚点（Anchor 工具）
1. Anchor 工具下在 Semitone 轨双击插锚 → 锚点值即整数（锚点值输入框显示整数）。
2. 拖动锚点上下移动 → 锚点逐格跳整数位，拖动中定位不抖、不来回跳变；
   松手后锚点输入框值为整数。
3. 框选一片锚点 → 选框跟手（不跳格），视觉框住的锚点即被选中。

### C. 整数滑条属性 lane
1. 侧栏 note 属性面板拖 **Steps (Int)** 滑条 → 吸附整数（原有行为，作对照基线）。
2. 把 **Steps (Int)** 钉上参数面板，在 lane 上拖写扫过若干 note → 各 note 段的顶线只落整数格位，
   与侧栏滑条读数一致（选中 note 对照侧栏面板值）。
3. 对照：钉 **tension**（线性滑条）lane 拖写 → 行为与改动前一致（连续）。

### D. 分段轨与回显轨回归（线性不变）
1. **Bend**（分段轨）绘制/擦除/锚点移动 → 与改动前一致（连续取值、段间断开）。
2. **Energy** 回显轨（合成后出现）显示位置正确（半透明面积、量程 0..100 不变形）。

### E. 存盘/撤销回归
1. 在 Semitone 轨画好台阶曲线 → Ctrl+Z 逐步撤销正常、重做恢复；存盘重开工程曲线原样。

### F. 对数轴（非线性 Custom 标度）
1. 参数栏切到 **LogFreq (Hz)**，上下界标签应显示 `20000 Hz`（顶）/ `20 Hz`（底）。
2. Anchor 工具在面板**竖向正中**双击插一个锚点 → 值约 **632 Hz**（几何中点 20·√1000），
   而非算术中点 10010——证明落点经 `ToValue` 走对数标度。
3. 拖动手感：锚点在**低频段**（面板下部）竖移，Hz 变化慢；在**高频段**（上部）竖移，Hz 变化快
   （等距屏幕位移 = 等比频率）。
4. 锚点值输入框键入 `1000` → 锚点跳到 1000Hz 对应的对数高度，显示回读 `1000 Hz`。
5. 连续标度不落格：曲线平滑（非阶梯），中间取值任意——与 Semitone 的阶梯对照，确认量化只对离散标度生效。
6. 拖到顶/底外侧 → 值钳在 20000 / 20，不溢出。

> 说明（非缺陷）：Hermite 插值在**值域（Hz）**进行，标度只作用于落点/显示/出口约束。故两个**稀疏**锚点之间
> 的曲线是 Hz 值域插值，映射到对数 Y 上会显得弯、时间中点值是 Hz 算术插值而非几何均值；freehand 密集绘制
> 贴合鼠标轨迹看不出差别。若需"屏幕域/几何插值"（log 扫频呈屏幕直线）是另一种约定，当前刻意不采用。
